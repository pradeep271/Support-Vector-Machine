# Support Vector Machines Tutorial  
## Kernel Selection and Hyperparameter Tuning

This repository contains a complete tutorial on **Support Vector Machines (SVMs)**, focusing on how **kernel choice and hyperparameter tuning** affect model performance and generalization.

---

## 📌 Project Overview

Support Vector Machines are powerful supervised learning models used for classification tasks. This project demonstrates:

- Differences between **Linear, Polynomial, and RBF kernels**
- The effect of **hyperparameters (C and gamma)**
- How to use **GridSearchCV for model optimization**
- Visualization of:
  - Decision boundaries
  - Accuracy comparison
  - Hyperparameter heatmaps

---

## 🎯 Objectives

- Understand how SVM works conceptually and practically  
- Compare kernel performance visually and quantitatively  
- Analyze overfitting vs underfitting  
- Learn how to tune models for better generalization
